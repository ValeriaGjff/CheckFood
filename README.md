# CheckFood

## CheckFood – это мобильное приложение, которое позволяет пользователю определить состав продукта, отсканировав штрих-код товара или ввести его вручную, и приложение покажет состав продукта, выделит вредные вещества, сохранит результат в историю и позволит добавить продукт в избранное.
### Основная цель приложения – помочь людям осознанно выбирать продукты, заботиться о здоровье, избегать вредных добавок и контролировать рацион.

## Описание приложения
### CheckFood – Android-приложение, созданное в Android Studio на Java.
Оно полностью повторяет дизайн и функциональность, предусмотренные макетом Figma.
Основные функции:
- Сканирование штрих-кода камерой
- Распознавание штрих-кода по фото из галереи
- Ручной поиск по коду
- Проверка состава и выделение вредных веществ
- Избранные продукты
- История проверок
- Настройки
- Экран "О приложении"

## Приложение использует:
- CameraX для работы с камерой
- ML Kit Barcode Scanning для распознавания кодов
- SQLite (локальная база) для хранения истории и избранного
- OpenFoodFacts API для получения состава продуктов

## Запуск на устройстве
1. На главном экране нажмите "Сканировать", наведите камеру на штрих-код
2. После сканирования откроется страница продукта: посмотреть состав, увидеть подсветку вредных веществ, добавить продукт в избранное.
3. В разделе “История” можно посмотреть предыдущие проверки
4. В разделе “Избранное” ваши сохранённые продукты
5. В разделе “Настройки” можно выбрать язык приложения
